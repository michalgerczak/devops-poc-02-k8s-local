# [DevOps Academy V1] Project 2: Kubernetes (Local)

This project builds on Project 1 by deploying the Dockerized 'hello-world' app to a local Kubernetes (Minikube) cluster.
It demonstrates hands-on skills in container orchestration, YAML manifests, and cluster management.

## Project Goal
Show practical understanding of Kubernetes fundamentals by writing and applying:
* A `Deployment` (to define and scale app replicas)
* A `Service` (to expose the app to the network)

## Key Consepts Demonstrated
* **Minikube:** Creating a lightweight, local Kubernetes cluster.
* **kubectl:** The CLI tool used to interact with the cluster.
* **Deployment:** Describing the *desired state* of an application (what to run and how many replicas).
* **Service:** *Exposing* the application with a stable network address.

## IBM Modules Applied
* **Introduction to Containers with Docker, Kubernetes and OpenShift:** Used for K8s architecture, objects (`Deployment`, `Service`), and `kubectl` workflow.
* **Hands-on Introduction to Linux Commands and Shell Scripting:** Will be used for a planned automation script (`helper.sh`).
